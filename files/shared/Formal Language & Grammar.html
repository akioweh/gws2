Formal Language & Grammar

A Language LLL is a set of finite-length strings
L⊆Σ∗L \subseteq \Sigma{}^*L⊆Σ∗

where Σ\SigmaΣ is its alphabet/vocabulary, the set of atomic symbols allowed in the language.
∗{}^*∗ is the Kleene Star operator1.

In an analogy of textual languages,
Σ\SigmaΣ is the set of all letters in the language, and
LLL is the set of all strings (words/sentences/paragraphs) that are grammatically correct.

To precisely define the contents of LLL, a language also has some formation rules. These rules govern how symbols may be concatenated into well-formed (allowed) strings;
L={w∈Σ∗∣w is well-formed}L = \{ w \in \Sigma{}^* \mid w \ \text{is well-formed} \}L={w∈Σ∗∣w is well-formed}

One formal way to define well-formed-ness is via a Grammar.

Alphabet / Vocabulary

Σ\SigmaΣ is the set of all (atomic) symbols allowed, divisible into logical symbols σ\sigmaσ and non-logical symbols μ\muμ:
Σ=⟨σ,μ⟩ \Sigma = \braket{\sigma, \mu} Σ=⟨σ,μ⟩

Logical symbols have constant meanings defined by the language syntax.
Non-logical symbols are analogous to variables; their meanings are semantic and given by the model/structure/interpretation of a formula.

Grammar

A Grammar is a specialized semi-Thue string rewriting system.

A Grammar is GGG is a 4-element tuple
G=⟨N,Σ,P,S⟩ G = \braket{N, \Sigma, P, S} G=⟨N,Σ,P,S⟩

  • NNN – Nonterminal symbols
    finite set disjoint from L(G)L(G)L(G)2

  • Σ\SigmaΣ – (the alphabet) Terminal symbols
    finite set disjoint from NNN

  • PPP – Production Rules
    finite set of “rules” in the form of
    (Σ∪N)∗N(Σ∪N)∗→(Σ∪N)∗(\Sigma \cup N) {}^* N(\Sigma \cup N) {}^* \rightarrow (\Sigma \cup N) {}^*(Σ∪N)∗N(Σ∪N)∗→(Σ∪N)∗

    a distinguished symbol in NNN
    functions as the starting point for the application of Production Rules

Other Remarks

Intuitively, NNN are abstract syntactic structures which “expand”, via steps in PPP, to a string of concrete alphabet symbols (Σ\SigmaΣ). SSS is the origin of the expansion.
Then, a sentence is a “full” expansion (which can expand no more) and LLL is all such full expansions.

If you know what an AST is, SSS is the root of the tree, NNN are subtrees (or nodes that denote structural sections), and Σ\SigmaΣ are the nodes. Then PPP is just the rules for which nodes and how nodes can be each others’ children.

Note on Grammar vs Language:

Grammar and Language do not form a one-to-one relationship.

A language defines a set of concatenations of Terminal symbols as valid.
A grammar derives this set from “rules”. (Remember a grammar is an elaborate set-builder.)

As such, a language may be represented by (derived from) a grammar, but not vice-versa, since:
!: A language could be represented with more than one different (classes of) grammars.

E.g., for the “regular” classification:
A language is regular if it can be generated by a regular grammar; regular grammars always generate regular languages.
