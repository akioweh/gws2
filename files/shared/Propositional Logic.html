Propositional Logic

Propositional Logic is a Formal System – a Language (syntax) and a set of Deduction Rules (semantics) – capable of expressing “propositional logic”.

It is one of the more primitive / basic systems for logic.

Unlike First-order Logic, there is one fixed language1 for propositional logic, i.e., authors do not generally define custom objects (in addition to those below) to use within formulas.

Well-Formed Formulas

Aka., Sentential Formulas, or the Language in Formal Language & Grammar

The set of Well-Formed Formulas (WFF)s, L\mathcal{L}L, is defined inductively:

  1. Base: P⊂L\mathcal{P} \subset \mathcal{L}P⊂L.
  2. Negation: φ∈L  ⟹  ¬φ∈L\varphi \in \mathcal{L} \implies \neg \varphi \in \mathcal{L}φ∈L⟹¬φ∈L.
  3. Binary Connectives: φ,ψ∈L  ⟹  (φ∧ψ),(φ∨ψ),(φ→ψ),(φ↔ψ)∈L\varphi, \psi \in \mathcal{L} \implies (\varphi \land \psi), (\varphi \lor \psi), (\varphi \to \psi), (\varphi \leftrightarrow \psi) \in \mathcal{L}φ,ψ∈L⟹(φ∧ψ),(φ∨ψ),(φ→ψ),(φ↔ψ)∈L
  4. Closure: No other strings are in L\mathcal{L}L.

where φ\varphiφ and ψ\psiψ represent strings, and everything else are strings of symbols as-is.

In actual writing, some parentheses are omitted for brevity.
To maintain unambiguity in how unparenthesized sub-formulas are parsed, the connectives are assigned precedence levels. In descending order:

  1. ¬\lnot¬ (highest)
  2. ∧\land∧ and ∨\lor∨
  3. →\to→ and ↔\leftrightarrow↔ (lowest)

The binary connectives are left associative, while negation is right associative.

Semantics

Truth Assignment

Aka. Valuation

A truth assignment for a formula maps each proposition variable in the formula to a truth value, then evaluates the formula.

I.e., a truth assignment is a function
ρ:P→{⊥,⊤} \rho: \mathcal{P} \to \{ \bot, \top \} ρ:P→{⊥,⊤}

where ⊤\top⊤ represents truth and ⊥\bot⊥ represents falsity.

Then, ρ\rhoρ is inductively defined for all sentential formula.
For φ,ψ∈L\varphi, \psi \in \mathcal{L}φ,ψ∈L,
ρ(¬φ)  ⟺  ¬ρ(φ)ρ(φ∧ψ)  ⟺  ρ(φ)∧ρ(ψ)ρ(φ∨ψ)  ⟺  ρ(φ)∨ρ(ψ)ρ(φ→ψ)  ⟺  ¬ρ(φ)∨ρ(ψ)ρ(φ↔ψ)  ⟺  ρ(φ)↔ρ(ψ) \begin{align*} \rho(\lnot \varphi) &\iff \lnot \rho(\varphi) \\ \rho(\varphi \land \psi) &\iff \rho(\varphi) \land \rho(\psi) \\ \rho(\varphi \lor \psi) &\iff \rho(\varphi) \lor \rho(\psi) \\ \rho(\varphi \to \psi) &\iff \lnot \rho(\varphi) \lor \rho(\psi) \\ \rho(\varphi \leftrightarrow \psi) &\iff \rho(\varphi) \leftrightarrow \rho(\psi) \\ \end{align*} ρ(¬φ)ρ(φ∧ψ)ρ(φ∨ψ)ρ(φ→ψ)ρ(φ↔ψ)​⟺¬ρ(φ)⟺ρ(φ)∧ρ(ψ)⟺ρ(φ)∨ρ(ψ)⟺¬ρ(φ)∨ρ(ψ)⟺ρ(φ)↔ρ(ψ)​

where the following (very well-formatted) truth tables define the logical connectives. Note that IMPLIES, →\to→, is defined in terms of ¬\lnot¬ and ∨\lor∨.

NOT:

¬\lnot¬
⊤\top⊤ ⊥\bot⊥
⊥\bot⊥ ⊤\top⊤

AND:

∧\land∧ ⊤\top⊤ ⊥\bot⊥
⊤\top⊤ ⊤\top⊤ ⊥\bot⊥
⊥\bot⊥ ⊥\bot⊥ ⊥\bot⊥

OR:

∨\lor∨ ⊤\top⊤ ⊥\bot⊥
⊤\top⊤ ⊤\top⊤ ⊤\top⊤
⊥\bot⊥ ⊤\top⊤ ⊥\bot⊥

EQUIV:

↔\leftrightarrow↔ ⊤\top⊤ ⊥\bot⊥
⊤\top⊤ ⊤\top⊤ ⊥\bot⊥
⊥\bot⊥ ⊥\bot⊥ ⊤\top⊤

Logical Classification:

  • Tautology: φ\varphiφ is a tautology if ρ(φ)\rho(\varphi)ρ(φ) is true for every truth assignment ρ\rhoρ.
  • Contradiction: φ\varphiφ is a contradiction if ρ(φ)\rho(\varphi)ρ(φ) is false for every truth assignment ρ\rhoρ.
  • Contingency: φ\varphiφ is contingent if it is neither a tautology nor a contradiction.

Rules of Inference

By defining the behavior of (a sufficient subset of)2 the logical connectives like above, the logic is already complete.
Alternatively, Rules of Inference (axioms) are defined, from which the intuitive behavior of the logical connectives are inferred.

The minimal set as Rules of Inference is just Modus Ponens (plus some form of definition for negation or NOT).

Modus Ponens (MP):

φ,φ→ψψ \frac{\varphi,\quad \varphi \to \psi}{\psi} ψφ,φ→ψ​

I.e., “if φ\varphiφ is true, and it is asserted φ→ψ\varphi \to \psiφ→ψ, then ψ\psiψ must be true”.

Other common Axioms

Hilbert-style:

  1. φ→(ψ→φ)\varphi \to (\psi \to \varphi)φ→(ψ→φ)
  2. (φ→(ψ→χ))→((φ→ψ)→(φ→χ))(\varphi \to (\psi \to \chi)) \to ((\varphi \to \psi) \to (\varphi \to \chi))(φ→(ψ→χ))→((φ→ψ)→(φ→χ))
  3. (¬φ→¬ψ)→((¬φ→ψ)→φ)(\neg \varphi \to \neg \psi) \to ((\neg \varphi \to \psi) \to \varphi)(¬φ→¬ψ)→((¬φ→ψ)→φ)

There are many axiomatic systems for propositional logic. As long as they all establish propositional logic, they’re all equivalent (obviously?!).

Meta-Theoretical Properties

These aren’t specific to propositional logic, but rather of any logic in general.

  • Soundness
    If ⊢φ\vdash \varphi⊢φ (i.e., φ\varphiφ is derivable), then φ\varphiφ is a tautology.
  • Completeness
    If φ\varphiφ is a tautology, then ⊢φ\vdash \varphi⊢φ.
  • Decidability
    A truth table construction provides an effective decision procedure for determining whether a given formula is a tautology.
  • Compactness
    A set of formulas is satisfiable if and only if every finite subset is satisfiable.

While (all) these properties may seem trivial, e.g., how soundness would not imply completeness, they aren’t.
